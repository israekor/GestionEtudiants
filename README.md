# Application de Gestion des Étudiants – Laravel 12

## Description

Application web permettant la gestion des étudiants avec authentification
classique et connexion Google via Firebase.

## Fonctionnalités

-   Authentification (Login / Logout)
-   Connexion Google (Firebase OAuth 2)
-   Gestion des rôles (Admin / Étudiant)
-   CRUD des étudiants (Admin)
-   Profil étudiant

## Technologies

-   Laravel 12
-   PHP 8.x
-   MySQL
-   Firebase Authentication
-   Tailwind CSS

## Installation

1. Cloner le projet
2. Installer les dépendances
3. Configurer le fichier .env
4. Lancer les migrations
5. Démarrer le serveur

⚠️ Le fichier de configuration Firebase (`firebase.js`) n’est pas versionné
pour des raisons de sécurité.  
Chaque utilisateur doit créer son propre fichier à partir de Firebase Console.
